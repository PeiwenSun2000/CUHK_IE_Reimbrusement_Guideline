Post-trip reimbursement example folder. Required for **all** reimbursement cases. Submit by **email** to **Fandy**.

| File | Description |
|------|-------------|
| `1_reciept.pdf` | Flight receipt |
| `1_payment_record.pdf` | Flight payment record |
| `2_registration-receipt.pdf` | Registration fee receipt |
| `2_Payment_record.pdf` | Registration payment record |
| `3_hotel_receipt.pdf` | Hotel receipt (if applicable) |
| `3_payment_record.pdf` | Hotel payment record (if applicable) |
| `4_acc_letter.pdf` | Paper abstract / supporting document |
| `5_first_page.pdf` | Acceptance notification |
