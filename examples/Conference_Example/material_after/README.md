# material_after

出行后报销材料示例目录。**所有报销形式均需提交**，整理后**邮件**发送给 **Fandy**。

按数字前缀命名，放入对应 PDF：

| 文件名 | 说明 |
|--------|------|
| `1_reciept.pdf` | 机票收据 |
| `1_payment_record.pdf` | 机票付款记录 |
| `1_boarding_pass.pdf` | **登机牌**（电子登机牌 App 截图或纸质登机牌扫描；需含乘客姓名、航班号、出发/到达日期与时间；多段航程可合并为一个 PDF） |
| `2_registration-receipt.pdf` | 注册费收据 |
| `2_Payment_record.pdf` | 注册费付款记录 |
| `3_hotel_receipt.pdf` | 酒店收据（如有） |
| `3_payment_record.pdf` | 酒店付款记录（如有） |
| `4_acc_letter.pdf` | 论文摘要 / 支持材料 |
| `5_first_page.pdf` | 论文接收通知 |

> **关于 `1_boarding_pass.pdf`：** 本示例目录中该文件**尚未放入**；出行后报销机票时请务必自行补充，与 `1_reciept.pdf`、`1_payment_record.pdf` 一并邮件提交给 Fandy，作为实际乘机的辅助证明。

详见 [README.md](../../README.md)（主文档「出行后材料」一节）。
