# material_after

出行后报销材料示例目录。**所有报销形式均需提交**，整理后**邮件**发送给 **Fandy**。

按数字前缀命名，放入对应 PDF：

| 文件名 | 说明 |
|--------|------|
| `1_reciept.pdf` | 机票收据 |
| `1_payment_record.pdf` | 机票付款记录 |
| `2_registration-receipt.pdf` | 注册费收据 |
| `2_Payment_record.pdf` | 注册费付款记录 |
| `3_hotel_receipt.pdf` | 酒店收据（如有） |
| `3_payment_record.pdf` | 酒店付款记录（如有） |
| `4_acc_letter.pdf` | 论文摘要 / 支持材料 |
| `5_first_page.pdf` | 论文接收通知 |

详见 [README.zh-CN.md](../../README.zh-CN.md)。
